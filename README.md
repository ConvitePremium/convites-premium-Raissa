# Raissa - 15 anos

Convite Premium Engine V1 corrigido.

## Publicar
1. Crie um repositório público no GitHub, sugestão: `convites-premium-Raissa`.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Ative o GitHub Pages: Settings > Pages > Deploy from branch > main > /(root).
4. Link final: `https://convitepremium.github.io/convites-premium-Raissa/`

## Testar botões
Use `?debug=1` no final do link para visualizar as áreas clicáveis.
