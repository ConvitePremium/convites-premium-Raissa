# Raissa - 15 anos

Convite Premium Engine V1 otimizado.

## Como publicar
1. Crie um repositório público no GitHub, sugestão: `raissa-15`.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Ative GitHub Pages em Settings > Pages > Deploy from branch > main > /(root).
4. O link ficará: `https://convitepremium.github.io/raissa-15/`

## Teste de botões
Use `?debug=1` no final do link para visualizar as áreas clicáveis.
